# Доступний Дім

Платформа підбору та аудиту житла для людей на кріслах колісних.

## Стек

- Django 5.x, Python, SQLite (розробка), Bootstrap 5.

## Швидкий старт (Windows)

**Найпростіше:** подвійний клік по файлу **`run.bat`** в корені проєкту. Скрипт сам створить venv, встановить залежності, застосує міграції і запустить сервер.

Потім у браузері відкрийте: **http://127.0.0.1:8000/**

---

Або вручну в терміналі (обовʼязково з папки проєкту):

```powershell
cd C:\Users\asus\OneDrive\Desktop\Duplom_dominik
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Відкрийте: **http://127.0.0.1:8000/**

**База даних:** зберігається в папці проєкту `data/db.sqlite3`.

## Структура

- **Головна** — пошук житла з фільтрами: ширина ліфта > 80 см, душ-трап, відсутність порогів.
- **Сторінка об'єкта** — таблиця аудиту: критерій | бал | статус | коментар; кольорові бейджі (9–10 зелений, 6–8 жовтий, <6 червоний).
- **Форма аудитора** — введення результатів перевірки з валідацією замірів у см.

## Адмінка

```bash
python manage.py createsuperuser
```

Потім: http://127.0.0.1:8000/admin/ — додавання об'єктів і перегляд аудитів.

## Логіка статусів за балами (1–10)

- **9–10** — Ідеально (зелений)
- **6–8** — Потребує покращення (жовтий)
- **3–5** — Важкодоступно (червоний)
- **1–2** — Недоступно (червоний)

`total_score` рахується автоматично як середнє значення балів (вхід, ліфт, пороги, наявність радіусу розвороту).
